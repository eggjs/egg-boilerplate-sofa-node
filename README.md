## Installation

```bash
$ npm i -g egg-init
$ egg-init --package sofanode sofanode-test
$ cd sofanode-test
$ npm i
$ npm run proxy
$ npm run test
```

See [egg-init](https://github.com/eggjs/egg-init) for more detail.
